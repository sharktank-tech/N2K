# N2K Ateliê

Projeto freelancer de um site especializado em personalização, consertos e design de roupas.

## características

- *Estrutura simples*
- *Pagina de Sobre*
- *Pagina de Políica de Privacidade*
- *Pagina de catálogo*

## tecnologias utilizadas

- *Backend:* Python (Flask)

- *Banco de Dados:* SQLite
